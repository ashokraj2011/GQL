Tech Notes & QA Coverage

Architecture decisions, NFRs, test coverage and open conflicts.

Architecture Notes
Component separation: Rendering, eligibility retrieval and analytics as separate concerns to reduce blast radius.
Resilience: Timeout and stale-cache fallback required for home-page widget SLA compliance.
ADRs: 4 recorded — offer rendering, cache TTL, API circuit breaker, telemetry schema.
Observability: Pending — may be gated to second release if scope tightens.
Non-Functional Requirements

Latency SLA: Offer widget must render within 300ms of page load (P95).

Availability: Eligibility timeout must not cause page-level failure.

Cache TTL: Configurable per segment. Default pending Architecture confirmation.

Security: PII in eligibility response must not be logged or exposed client-side.
